
para verificar o funcionamneto:
git status

para atualizar todas as modificações:
git add .


Fazer o commit:

git commit -m"O que foi adicioado na versão"

Enviar commit:
git push

Vê histótico das versões do meu código:
git refolg

navegar entre versões do meu código(voltar a versão desejada):
git reset --hard "endereço da versão que se encontra dando um reflog(não usa aspa)"

navegar entre versões do meu código(voltar a versão atual):
git reset --hard "endereço da versão atual que se encontra dando um reflog(não usa aspa)"

como vê as suas branch(Onde fica salvo as versões):
git branch

como criar uma branch:
git branch staging(ou qualquer outro nome)

como mudar de branch:
git checkout staging (nome da branch)


como colocar as atualizações na branch master(ou a que preferir):

1° passo-> tem que colocar na branch principal:
git checkout master(ou a que preferir)

2° passo -> trazer as atualizações do servidor para a máquina local:
git pull

3°passo -> unir os códigos:
git merge (nome da branch de onde quero puxar as atualizações)

4° passo-> enviar atualização para a branch no servidor:
git push
